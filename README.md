🪨📄✂️ Rock Paper Scissors Game (Python)

A simple command-line Rock Paper Scissors game built using Python. This project is great for beginners to practice concepts like loops, conditionals, user input, and the random module.

🎮 How the Game Works

The player chooses one of:

ROCK

PAPER

SCISSORS

The computer randomly selects an option.

The winner is decided based on the rules:

Rock beats Scissors

Scissors beats Paper

Paper beats Rock

The game continues until the player chooses to exit.

🚀 Features

Random computer choice using Python’s random module

Continuous gameplay loop

Input validation

Simple and interactive terminal interface

🛠️ Technologies Used

Python (Core Concepts)

Variables

Loops (while)

Conditional Statements (if-elif-else)

Functions from random module

▶️ How to Run

Make sure Python is installed on your system.

Save the code in a file named:

rps_game.py

Open terminal / command prompt and run:

python rps_game.py
